# Mot Mystère

## Description

Un jeu de devinette de mots basé sur une base de données de 22 000 mots français. Le jeu évalue la difficulté de chaque mot et propose plusieurs modes de jeu: **Facile**, **Normal**, **Expert** et **Mort Subite**.


## Installation
1.  Installe les dépendances :

    `pip install -r requirements.txt`

2.  Exécute le fichier :

    `main.py`

## Modules
MBrick utilise 3 modules:
* **CustomTkinter**: pour l'interface
* **Pandas**: pour la gestion de la base de données de mots et du classement
* **Matplotlib**: pour le graphique
